How to work on Linux:
1. Run the run executable
2. Follow the menu
3. You must enter the numbers in the menu, for example:
'1' --- Create password database
Select action: 1 + 'Enter'
Enter password database name
etc.

How to work on Android:
1.Install termux
2.In termux install python3 and required libraries
3.Next, place the executable files and folders in a place convenient for you
4. By analogy with Linux in termux, run the executable file run
5.Follow the menu


Additional information:
Executable file - "run.py"
Program icon (optional) - "KeePassIc.ico"
The "bases" folder is required to store password databases
"img" folder is required for executable files
The "keys" folder is required to work with encryption key files (optional)
