Как работать на Linux:
	1. Запустите исполнительный файл run
	2. Следуйте указаниям меню
	3. Цифры в меню вы должны вводить,например:
		'1' --- Создать базу паролей
		Выберите действие: 1 + 'Enter'
		Введите имя базы паролей
		и т.д.

Как работать на Android:
	1.Установите termux
	2.В termux установите python3 и необходимые библиотеки
	3.Далее поместите исполнительные файлы и папки в удобное для вас место
	4.По аналогии с Linux в termux запустите исполнительный файл run
	5.Следуйте указаниям меню


Доп.информация:
	Испольнительный файл - "run.py"
	Иконка программы (по желанию) - "KeePassIc.ico"
	Папка "bases" необходима для хранения баз паролей
	Папка "img" необходима для испольнительных файлов
	Папка "keys" необходима для работы с файлами ключей шифровани(по желанию)
